# dci-data

Public **data feed** for the Downbeat Designs DCI scores pages — score JSON only.
The scraper that produces this data is kept private (not in this repo).

The website reads these files via:
  https://raw.githubusercontent.com/downbeatdesigns/dci-data/main/data/<file>.json

## Files
- `data/dci_scores_2025.json` — complete 2025 DCI season (all events, all divisions).

## Notes
- For the 2026 season, fresh JSON will be published here automatically by a private job.
